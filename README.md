# Página de Receitas 

Projeto simples desenvolvido para praticar **HTML** e **CSS**.  
A página apresenta uma receita de **Cupcake de café com chantilly**, com imagem, ingredientes e modo de preparo.

##  Tecnologias
- HTML5
- CSS3
- Google Fonts (Alice)

##  Estrutura do projeto
- index.html
- style.css
- assets/
- bg-image.jpg
- main-image.jpg

##  Como visualizar
1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em qualquer navegador.

##  Resultado
Uma página estática simples, com layout agradável e estilização básica, servindo como exercício de aprendizado.
